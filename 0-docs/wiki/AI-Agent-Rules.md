# AI Agent Rules — 5amClub Workspace

These rules apply to all AI assistants connected to this repo:

- ChatGPT

- Claude

- Cursor

- Perplexity

- Any agent orchestrators

## 1. General Behavior

- Always respect the folder structure.

- Always store new content in the correct directory.

- Keep all generated files clean, readable, and organized.

- Follow README instructions for each directory.

- Ask for clarification only when necessary.

## 2. File Storage Rules

- Coursework → `/2-coursework/<Category>`

- Homework → `/4-homework/<Category>`

- Cheatsheets → `/3-cheatsheets/<Category>`

- Engineering Projects → `/5-engineering-projects/<Project>`

- Resources → `/8-resources`

- Top-level documentation → `/0-docs`

## 3. Content Requirements

- Include explanations, steps, or reasoning when appropriate.

- Use fully annotated and formatted Markdown.

- When generating code, include comments.

- When writing math, use LaTeX formatting.

- When summarizing PDF or textbook content, cite the location/page.

## 4. Naming Conventions

- Use lowercase-kebab-case for files:

  - `laplace-transform-summary.md`

  - `week3-homework-solutions.md`

- Use uppercase folder names only where predefined in the repo.

## 5. Style Guide

- Keep tone: clear, direct, helpful, engineering-focused.

- Avoid filler.

- Prefer precision and structure.

- Use section headers for readability.

- Include diagrams in ASCII or Markdown when useful.

## 6. Boundaries

AI agents should NOT:

- Generate harmful content.

- Write grading answers that violate academic integrity.

- Replace original student work when prohibited.

## 7. Primary Objectives

1. Support learning and mastery.

2. Support engineering creativity.

3. Support personal productivity and automation.

4. Maintain the integrity and longevity of the workspace.

## 8. Update Rules

Agents may:

- Update existing documentation if instructed.

- Append new notes.

- Improve clarity.

- Create new directories only if they match the system architecture.

## 9. Error Handling

If unsure where a file goes:

- Store it under `/9-archive` for manual sorting.

- Leave a note explaining the decision.

---

This file ensures all agents behave consistently, intelligently, and safely within the workspace.
