# 5amClub OS v6.1.0

## Deployment

### Vercel (Recommended)
1. Connect this repository to [Vercel](https://vercel.com)
2. Set Root Directory to: `jonny study` (if files are in subfolder) or `.` (if in root)
3. Deploy!

### Netlify (Alternative)
Drag and drop the `jonny study` folder into [Netlify](https://app.netlify.com/drop).

## Features
- **Massive Resource Database**: 100+ consolidated resources for SE, CS, Neuro, Math, Chem, and EE/BME.
- **Notion-Style Planner**: Drag and drop weekly planning with per-day task buttons.
- **Project Gallery**: Visual project cards with image support.
- **Smart Resource Linker**: Link existing resources to courses or create new ones.
- **Course Notes**: Dedicated notes for each course with Markdown support.
- **Progress Tracking**: Visual progress bars and dashboard charts.
- **Mobile Optimized**: Full sidebar navigation on mobile.
